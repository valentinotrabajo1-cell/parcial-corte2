El programa tiene varias mejoras que van más allá de lo básico del proyecto. Una de ellas es que no todos los espacios de parqueo son iguales, ya que se incluyeron zonas especiales para motos y para personas con discapacidad. Esto hace que el parqueadero sea más parecido a uno real.

También se usaron colores en la consola para mostrar el mapa, lo que ayuda a entender mejor qué espacios están libres, ocupados o son de otro tipo. Esto hace que el programa sea más fácil de usar y más visual.

Otra mejora importante es que el sistema puede asignar un espacio automáticamente al vehículo, sin que el usuario tenga que elegirlo. Esto facilita el proceso y además se hizo usando punteros, cumpliendo también con un requisito del proyecto.

El programa también evita que el usuario parquee en lugares incorrectos como las vías, la entrada o la salida, lo que ayuda a que no haya errores.

Además, el cobro se hace por minutos y no por horas, lo que lo hace más preciso y más realista. Para esto se usa la hora del computador.

Por último, se agregó una opción para ver estadísticas, donde se muestra cuántos espacios están ocupados y cuántos están libres.

En general, estas mejoras hacen que el programa sea más completo, más fácil de usar y más parecido a un sistema real.
